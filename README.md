# java-online-shopping-cart
A simple console-based E-Commerce application in Java using OOP concepts like inheritance, polymorphism, and collections. Users can view products, add items to cart, and checkout.


## How to Run This Project

### Prerequisites
- Java JDK 8 or above installed
- Git (optional, for cloning)

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/sachinprajapat09/java-online-shopping-cart.git
   
2. Navigate to the project directory:
   ```bash
    cd java-online-shopping-cart/OnlineShoppingCard

3. Compile the Java source files:
   ```bash
   javac com/shoppingcard/*.java
   
4. Run the application:
   ```bash
   java com.shoppingcard.MainApp
